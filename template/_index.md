# Vault Map

Personal second brain: projects, research, learning, creative work. Optimized for searchability and LLM navigation.

## Folder Structure

| Folder | Purpose |
|--------|---------|
| Journal/Daily | Daily notes (auto-created) |
| Journal/Weekly | Friday week-in-review |
| Journal/Monthly | Monthly rollup (last day) |
| Capture | Annoyances, Ideas (quick capture) |
| Projects | Project homepages |
| Creative | Novels (Longform) |
| Learning | Topics being learned |
| Research | OpenClaw research topics |
| Reference | Inventory, misc reference |
| Archive | Archived projects and learning |

## Naming Conventions

- Daily: `YYYY-MM-DD-Day` (e.g. 2026-02-25-Tue)
- Weekly: `YYYY-W##-Review`
- Monthly: `YYYY-MM-Monthly`
- Projects: `lowercase-kebab-case`

## Frontmatter Types

- `daily`, `week-review`, `monthly`, `project`, `learning`, `research`, `reference`, `creative`
- All notes use `tags: [..., YYYY-MM]` for monthly searchability

Start here. Follow links to section indexes (`Projects/_index.md`, etc.). Use `_index.md` files to discover content. Follow `[[links]]` to navigate.
