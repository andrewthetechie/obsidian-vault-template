# obsidian-vault-template
My personal obsidian vault template

## Quick Start

1. Create a vault in Obsidian
1. Copy all of the contents of the template folder into the vault
1. Install plugins: Templater, Periodic Notes, Rollover Daily Todos, Dataview, QuickAdd, Longform
1. Configure plugins: Use docs/plugin-config.md and docs/create-project-setup.md
1. Set up Periodic Notes - Point daily/weekly/monthly to the templates in Templates/
1. Create the QuickAdd macro - Follow docs/create-project-setup.md for the “Create Project” macro
1. Optional: migrate content

The vault is ready to use. Start with _index.md for an overview and links to the guides.
